# 3.5.2 Casos de prueba y resultados

> ⬜ **Pendiente de redacción.** **Cubre:** ISO/IEC 25010.
> **Guía:** [3-calidad-iso-25010.md](../3-calidad-iso-25010.md).

## Qué debe contener
- Tabla de casos de prueba (ID, Requisito, Precondición, Pasos, Resultado esperado, Resultado
  real, Estado) trazados a los RF/RNF de 2.3/2.4.
- Conteo real de pruebas por suite y su resultado (derivado de ejecutar las suites, no estimado).

## Comandos de demostración (sugeridos)
```bash
dotnet test apps/Auth/Test/*.csproj
dotnet test apps/Payment/Test/*.csproj
cd apps/Frontend && bun run test:ci
```
