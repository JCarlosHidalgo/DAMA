# 3.2.4 Requisitos no funcionales

> ⬜ **Pendiente de redacción.** **Cubre:** ISO/IEC/IEEE 29148 (RNF) + enlace con ISO 25010.
> **Guía:** [1-requisitos-iso-29148.md](../1-requisitos-iso-29148.md) y [3-calidad-iso-25010.md](../3-calidad-iso-25010.md).

## Qué debe contener
- RNF de seguridad (cifrado de contraseñas con rehash, JWT/RSA, autorización *default-deny*,
  aislamiento por `TenantId`), rendimiento, fiabilidad, mantenibilidad y escalabilidad.
- Cada RNF verificable; usar el catálogo OWASP como origen y evidencia.

## Comandos de demostración (sugeridos)
```bash
ls extra/OWASP/web-top-10-2021 extra/OWASP/api-security-top-10-2023
grep -rn "limit_req\|gzip\|client_max_body_size" infrastructure   # límites/rendimiento en gateway
```
