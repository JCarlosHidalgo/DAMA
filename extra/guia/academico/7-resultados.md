## Qué debe contener
- Síntesis de lo logrado frente a los objetivos: RF implementados y probados, características de
  calidad evaluadas, cobertura OWASP, sistema desplegado.
- Cierre de la trazabilidad objetivo → requisito → diseño → prueba → resultado.

## Comandos de demostración (sugeridos)
```bash
git rev-list --count HEAD                                   # volumen de trabajo
git log --pretty=format:'%ad' --date=short | sort | sed -n '1p;$p'   # ventana del proyecto
```
