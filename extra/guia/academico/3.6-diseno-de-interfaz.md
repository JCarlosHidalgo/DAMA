# 3.3.6 Diseño de interfaz (UI/UX)

> ⬜ **Pendiente de redacción.** **Cubre:** IEEE 1016 (diseño de interfaz) + ISO 25010:2023 (Interaction Capability).
> **Guía:** [2-diseno-ieee-1016.md](../2-diseno-ieee-1016.md) y [3-calidad-iso-25010.md](../3-calidad-iso-25010.md).

## Qué debe contener
- Wireframes/capturas de las pantallas principales por rol (login, dashboard, horario, pagos).
- Justificación de decisiones de usabilidad: tema claro/oscuro, diseño responsive (tablas que se
  apilan en móvil), identidad de marca, íconos FontAwesome.

## Comandos de demostración (sugeridos)
```bash
ls apps/Frontend/src/app                       # estructura de la SPA
cd apps/Frontend && bun run start              # levantar la SPA para capturar pantallas
```
